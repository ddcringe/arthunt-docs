# Пользовательский сценарий — Специалист

> **Статус:** Шаблон — требует заполнения  
> **Связанные сценарии:** AH-MVP-006, AH-MVP-007

## Описание роли

Специалист — фотограф, дизайнер, видеограф, иллюстратор или другой творческий профессионал, который размещает портфолио на платформе и ищет заказы.

## Основной путь (Happy Path)

### Шаг 1 — Регистрация и настройка профиля

**Триггер:** Специалист хочет найти заказы через платформу.

**Действия:**
1. Переходит на arthunt.ru и регистрируется через email или OAuth (VK/Google/Яндекс)
2. Подтверждает email (получает письмо с токеном)
3. Заполняет профиль: специализация, город, биография, ценовой диапазон
4. Добавляет теги (стиль, жанр, техника)

**Результат:** Профиль создан, но ещё не виден в поиске.

---

### Шаг 2 — Наполнение портфолио

**Триггер:** Профиль создан, нужно добавить работы.

**Действия:**
1. Загружает ≥3 работ в портфолио
2. Для каждой работы указывает: название, описание, теги, медиафайлы
3. Работы отправляются на модерацию

**Бизнес-правило:** Специалист появляется в поиске только после одобрения ≥3 работ.

**Результат:** Работы ожидают проверки модератора.

---

### Шаг 3 — Поиск проектов и отклик

**Триггер:** Портфолио прошло модерацию, специалист видит список проектов.

**Действия:**
1. Просматривает ленту активных проектов
2. Фильтрует по тегам, бюджету, срокам
3. Открывает интересный проект и читает описание
4. Отправляет отклик: сопроводительное письмо, предложенная цена, срок, ссылки на работы из портфолио

**Бизнес-правило:** 1 отклик на пару (специалист, проект).

**Результат:** Заказчик получает уведомление о новом отклике (AH-MVP-007).

---

### Шаг 4 — Получение приглашения

**Триггер:** Заказчик нашёл специалиста через поиск и отправил приглашение.

**Действия:**
1. Специалист получает email-уведомление (AH-MVP-006)
2. Открывает приглашение в личном кабинете
3. Принимает или отклоняет (с указанием причины)

**Результат:** При принятии — начало переговоров с заказчиком.

---

## Альтернативные сценарии

| Ситуация | Поведение системы |
|----------|-------------------|
| Работа отклонена модератором | Специалист получает уведомление с причиной отказа |
| Профиль заблокирован | Пользователь получает уведомление, профиль скрыт из поиска |
| Лимит откликов исчерпан | Система показывает предложение перейти на Premium |

## Связанные API эндпоинты

| Действие | Метод | Путь |
|----------|-------|------|
| Создать профиль | `POST` | `/specialists` |
| Обновить профиль | `PATCH` | `/specialists/{id}` |
| Добавить работу | `POST` | `/specialists/{id}/portfolio` |
| Список проектов | `GET` | `/projects` |
| Создать отклик | `POST` | `/projects/{id}/responses` |
| Мои приглашения | `GET` | `/specialists/{id}/invitations` |
